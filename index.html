Projects

QR Code Generator

Developed a QR code generator using Python, SQL, and the qrcode library.

  • A QR Code Generator is a software or tool that creates a Quick Response (QR) code, a type of two-dimensional barcode that stores information, such as text, URLs, contact details, or other data.
  • User Input: The user enters the desired data, such as text or a URL, into the QR Code Generator.
  • Data Encoding: The generator encodes the input data into a QR code format.
  • QR Code Generation: The encoded data is then used to generate a QR code image.
  • Output: The generated QR code image is displayed or saved for later use.
  • Input validation, multiple data types (text, URLs, contacts)
  • Customizable QR codes, stored in SQLite DB
  • Tools: Python, SQLite, qrcode, Pillow
  • Skills: Python, SQL, data validation, debugging, image generation
